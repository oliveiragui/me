DoubleThink!!

A game I did with my friend Peter for Projeto Integrador 3 (a class that seeks to bring out all the knowledge students learned and put into a single project) at uni. The goal here was to make a game from scratch using only java and its built-in packages, so we used java2d, swing and awt. We struggled a little with having to make our game without a library, so we followed this great tutorial from Marcus Man. Guidelines were to make the code using OOP, UML, activity diagrams, and make games based on atari/nes era. Me and Peter did the coding together, and I was most responsible for making the game design and logics. About the game, it's a top-down view based on classic japanese games, my idea was to make it a puzzle game where you have a clone player controlled simutaneously. The goal is to reach the top area of the level, for this you'll have to bring the both characters as close as possible to solve the puzzles and unlock the passage.

Stick & Go!!

Another game I did with my friend Peter for Projeto Integrador 2 at uni. On this project we had to make a game using Python and its PyGame library. It was our first game and we had much fun making it. The goal of this class was to make a simple 2D game and learn Python along the way. We tried to do something different and I came up with this idea of jumping through the levels to get to the objective. Peter coded most of the launching and jumping mechanic, I was responsible for designing and coding the levels. About the game, it's a game where everything is boxes, you are the red box, and have to jump to the portals, identified as pink boxes. The enemies are the moving boxes, and coins are the yellow boxes. It has 3 levels, and the screen selector is playable too. The gameplay is not that fluid, as we only had time to actually make it work rather than polish, due to our tight schedule of going to our job and then to the college. At the end of the day we enjoyed the outcome.
